## 슬라이드 8: 핵심 기술 4 - 다중 채널 힌트

### ControlNet을 위한 3채널 조건부 입력

```
┌─────────────────────────────────────────────────────┐
│  Red 채널: 결함 마스크 (지표 기반 향상)              │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━      │
│                                                     │
│  if Linearity > 0.7:                                │
│      Skeleton + Edge emphasis                       │
│      ━━━━━━━━━  (골격 + 경계선)                    │
│                                                     │
│  elif Solidity > 0.8:                               │
│      Filled mask                                    │
│      ████████  (채워진 마스크)                       │
│                                                     │
│  else:                                              │
│      Edge-emphasized                                │
│      ▓▓▓▓▓▓▓▓  (경계선 강조)                        │
└─────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────┐
│  Green 채널: 배경 구조선 (Sobel 엣지)                │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━      │
│                                                     │
│  vertical_stripe   → Sobel X                        │
│  horizontal_stripe → Sobel Y                        │
│  complex_pattern   → Sobel X+Y                      │
│  smooth/textured   → Minimal edges                  │
│                                                     │
│  [예시: 수직 줄무늬]                                 │
│  ║ ║ ║ ║ ║ ║  (수직 경계선만 강조)                  │
└─────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────┐
│  Blue 채널: 배경 텍스처 (로컬 분산)                  │
│  ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━      │
│                                                     │
│  Local Variance (7×7 window)                        │
│                                                     │
│  smooth         → Low values   ░░░░░               │
│  textured       → High values  ▓▓▓▓▓               │
│  complex        → Mixed        ░▓░▓░               │
└─────────────────────────────────────────────────────┘

    Red            Green           Blue
```
