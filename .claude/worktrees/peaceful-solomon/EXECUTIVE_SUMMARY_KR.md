## 1. 프로젝트 개요

### 1.1 비즈니스 과제

철강 제조 공정에서 표면 결함 탐지는 제품 품질과 고객 만족도에 직접적인 영향을 미칩니다. 현재 AI 기반 결함 탐지 시스템은 다음과 같은 문제에 직면하고 있습니다:

- **데이터 부족**: 희귀 결함 유형의 제한된 샘플 수
- **클래스 불균형**: 일부 결함은 전체의 10-15%에 불과
- **품질 편향**: 특정 배경 패턴에만 치중된 학습 데이터
- **개발 비용**: 새로운 데이터 수집 및 라벨링에 막대한 시간과 비용

### 1.2 솔루션

**컨텍스트 인식 철강 결함 증강(CASDA)** 시스템은 AI 기술을 활용하여 물리적으로 타당한 합성 결함 이미지를 자동 생성합니다. 이를 통해:

✅ **데이터 규모 16.5% 확대** (12,568개 → 14,643개)  
✅ **희귀 클래스 34.2% 증가** (Class 4)  
✅ **클래스 불균형 17.0% 개선**  
✅ **데이터 수집 비용 절감** (합성 데이터 활용)

## 2. 핵심 기술

### 2.1 시스템 아키텍처

CASDA는 5단계 자동화 파이프라인으로 구성됩니다:

```
1. ROI 추출 → 2. 학습 데이터 준비 → 3. 합성 이미지 생성 
   → 4. 품질 검증 → 5. 데이터셋 통합
```

### 2.2 기술적 차별점

| 기존 증강 기법 | CASDA 시스템 |
|---------------|-------------|
| 단순 회전/반전 | **물리 법칙 기반 생성** |
| 새로운 샘플 X | **완전히 새로운 샘플** |
| 제어 불가능 | **결함 위치/크기/형태 제어** |
| 배경 무시 | **배경-결함 최적 매칭** |

### 2.3 핵심 혁신

1. **4가지 통계 지표**: 결함의 기하학적 특성을 정량화
2. **배경 분석 시스템**: 텍스처 패턴 자동 분류
3. **매칭 알고리즘**: 물리적으로 타당한 결함-배경 조합
4. **품질 검증**: 5가지 메트릭으로 합성 품질 보장
